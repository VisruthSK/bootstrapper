# bootstrapper: Opinionated R Package Quickstart

Bootstrap new R packages with strongly opinionated defaults. Creates the
package scaffold, wires up tests, README and NEWS, configures GitHub
Actions and Dependabot, adds Air/Jarl setup, and drop in a default
AGENTS.md for building R packages.
